## 1. Introduction
This custom desktop PC build represents an **AMD-based extreme-entry level gaming** system designed for enthusiasts who demand premium performance without compromise. Built around the cutting-edge AMD Ryzen 7 9800X3D processor and MSI GeForce RTX 5080 graphics card, this configuration delivers exceptional gaming performance at 4K resolution while maintaining excellent frame rates across all modern AAA titles. The system features top-tier components including 64GB of DDR5-6000 RAM, PCIe 5.0 NVMe storage, and liquid cooling to ensure optimal performance under demanding workloads.

With a budget of approximately ₱210,000, this build embodies the philosophy of "To be wealthy! Nearly every option in this tier is 'best-in-class', and is priced to match." While this represents a significant investment that borders on fanaticism, each component has been carefully selected to provide unparalleled gaming experiences and future-proofing for years to come. This extreme-entry level approach ensures that users won't need to upgrade individual components for an extended period, making it a sound long-term investment for serious gaming enthusiasts.

## 4. Compatibility Justification
For our **AMD-based extreme-entry level gaming** build, each component represents best-in-class performance with perfect compatibility:

- **CPU + Motherboard**: AMD Ryzen 7 9800X3D (AM5) works seamlessly with MSI MPG X870E CARBON WIFI motherboard (AM5 socket, BIOS compatible). This flagship gaming CPU with 3D V-Cache delivers uncompromised performance.
- **RAM**: Corsair Vengeance RGB 64 GB DDR5-6000 utilizes motherboard's four DDR5 slots (up to 128 GB supported). Extreme-entry level capacity for demanding gaming scenarios.
- **CPU Cooler**: MSI MPG CORELIQUID K240 liquid cooler fits ATX case and motherboard bracket perfectly. Essential for maintaining flagship CPU performance under gaming loads.
- **Storage 1 (SSD)**: Samsung 990 EVO Plus 4 TB NVMe connects to motherboard's PCIe 5.0 M.2 slots. Premium storage with lightning-fast load times for AAA games.
- **Storage 2 (HDD)**: Toshiba N300 4 TB NAS HDD uses motherboard SATA 6G ports. Additional storage for extensive game libraries typical of extreme gaming enthusiasts.
- **Graphics Card**: MSI RTX 5080 16 GB fits comfortably in ATX case with no CPU bottleneck. Heart of extreme-entry level build delivering uncompromised 4K gaming with ray tracing.
- **Power Supply**: Asus ROG Strix 1000 W 80+ Platinum provides 1000W capacity (672W system requirement). Best-in-class modular PSU for premium component stability.
- **Case**: MSI MAG VAMPIRIC 100R ATX mid-tower accommodates all premium components with proper clearance and airflow.
- **Case Fans**: Corsair iCUE LINK LX120 RGB 3-pack supports 120mm mounting with PWM control. Premium cooling aesthetics matching build quality.
